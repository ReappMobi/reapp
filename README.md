## Description

[Reapp](https://github.com/ReappMobi/reappt) is a free, open-source social network server where users can follow and donate to social institutions discover new ones.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
